# Bank-loan-report-analysis-and-visualization(Interactive Dashboard creation using Tableau)
## Project objective
This project analyzes a bank loan dataset to understand key financial metrics, borrower trends, default patterns, and loan health. The insights are presented through an interactive Tableau dashboard designed for clear decision-making and business reporting
